To activate

  • Charge ≥ 50%, connect to Wi-Fi
  • Pick a plan (online, no store needed)
  • Enter IMEI → scan the QR code
iPhone 11 SIM gotcha: it uses the older eSIM type — it can't do eSIM Quick-Transfer/push. You must activate via the QR-code method, with Wi-Fi (or the old physical SIM still in) so it can download.

🎯 The pick

Just phone + normal data? → Metro $25. Cheapest unlimited anywhere, price locked 5 years, on T-Mobile's own network (no MVNO deprioritization). Since the phone's already a Metro device, it's the path of least resistance.

Tether a lot? → Straight Talk Gold $55 for 30GB hotspot, or Platinum $65 if you need truly unlimited hotspot.

Want freebies? → Metro $60 bakes in Amazon Prime (~$15/mo value).
