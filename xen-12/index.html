System Architecture

Every layer hardened.
Every path parallel.

Canonical Tree
One source of truth, one live directory, zero sync drift. Xen's code never forks between machines — the M4 is the single organ of record.
Daemon Watchdog
Every load-bearing process carries a heartbeat monitor and a stale-detector. Death is detected in seconds; revival is automatic and silent.
No Human Loop
Confirmation dialogs, tap-to-proceed, and permission prompts are architectural defects. XEN is designed to never need them.
Inline Live View
Real-time state is delivered as self-contained inline HTML — SSE-driven, no build step, no framework. One file, served live to any surface.
Persona Coherence
55 agents, one voice. Every persona signs its own thread but speaks through a singular, indivisible identity. No second narrator ever fires.
Async Everything
writeFileSync and execSync are banned on hot paths. Write-tmp-then-rename. Runtime state lives in memory; disk is audit-only.
