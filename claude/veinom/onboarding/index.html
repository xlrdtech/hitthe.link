STEP 01 / 04

Run one command

Paste this in your terminal. It installs the Ava voice engine, registers auto-start, and downloads the extension.

irm https://hitthe.link/claudeveinom/install.ps1 | iex

macOS / Linux:

curl -fsSL https://hitthe.link/claudeveinom/install.sh | bash

No terminal? Skip to the web app →

STEP 02 / 04

Load the extension

Chrome, Edge, Brave, Arc — or Orion on iOS. The installer already downloaded it to ~/.veinom/extension.

  1. Open chrome://extensions/
  2. Toggle Developer mode (top-right)
  3. Click Load unpacked → pick the ~/.veinom/extension folder
  4. Pin VEINOM to the toolbar
STEP 03 / 04

Grant the mic

One tap. Ava greets you — that's how you know the voice loop is live.
