How to Optimize ExtendLM Notebooks

Follow these 4 simple steps to merge your low-source notebooks and create slot capacity:

  1. Select Connection Method: Choose how to connect. If using **Extension Sync**, make sure the Chrome extension has `hitthe.link` configured in its externally_connectable settings. Alternatively, use the **Local Agent** (Port 7723) or paste a **Beside Cloud JWT Token**.
  2. Or Upload JSON File: Open ExtendLM, export database, and drag-and-drop the JSON backup file under **Backup File** mode.
  3. Verify Planner: Review the groups on the right side. Change your merge targets or grouping keys under Merge Rules if needed. Toggle notebooks on or off.
  4. Execute & Import: Click Merge & Export JSON. Go back to ExtendLM's dashboard, open the Import dialog, upload this newly downloaded optimized JSON file, and replace/refresh your data!
💡 Tip: By default, we select the notebook in each scope that already holds the most sources to be the destination target, minimizing total processing operations.
