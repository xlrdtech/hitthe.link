Auth

Everything above rides session cookies — no bearer token observed on claude.ai. A separate, real OAuth2 flow exists for the Claude Code CLI at api.anthropic.com/v1/oauth/token, documented for architecture literacy only — reusing it outside Anthropic's own client sits in ToS gray-zone territory per third-party research, not recommended without explicit go-ahead on that risk.
